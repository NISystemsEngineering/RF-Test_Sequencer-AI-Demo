# RF-Test_Sequencer-AI-Demo
C# based Test Program for a WLAN RF Amplifier that uses NI-RFmx and RFSG APIs to test multiple frequency bands, WLAN Standards, Channel Bandwidths and performs EVM, TxP, SEM measurements, logs the results to CSV and is capable to create a TestStand sequence.
